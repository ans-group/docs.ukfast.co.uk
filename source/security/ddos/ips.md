# IP Ranges

For some applications you may want to know the IP Ranges that are used on the DDoSX<sup>®</sup> platform (e.g. such as for defining a trusted list of IPs to accept X-Forwarded-For header from.)

## IPv4: 
### Unicast Range:
* 185.156.64.0/24
* 23.170.128.0/24
* 192.166.44.0/24
* 78.24.88.0/24
* 195.69.102.0/24

### Anycast Range:
* 185.181.196.0/22


## IPv6:
### Unicast Range:
* 2a02:21a8:1::/48
* 2a02:21a8:2::/48
* 2a09:ba00:4::/48
* 2a09:b600:5::/48
* 2a09:b200:6::/48

### Anycast Range:
* 2a02:21a8::/48
