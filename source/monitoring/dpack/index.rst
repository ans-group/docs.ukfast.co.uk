=========================================================
Using DPACK to monitor server/virtual machine performance
=========================================================

The Dell Performance Analysis Collection Kit (DPACK) is free software provided by Dell to check performance of a server/virtual machine for a set period of time. It is totally non-intrusive, so can be run on production environments at any time.
