# Git

### Install Git
You can install git with the following command:

```bash
~]# yum install git
```

#### Updating Git
You can update git with the following command:

```bash
~]# yum update git
```

#### Version Check
You can check the version of git installed with the following command:

```bash
~]# ~]# git --version
git version 1.8.3.1
```
